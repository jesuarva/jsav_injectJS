'injectJS' a library by jesuarva

injectJS crates an object in the window namespace called 'jsav_ijs'

'jsav_injectJS' fetch() JS libraries. Then injects each resource in the DOM by precedency

Methods:

Check console for more details.

load()

Injects a single library in the DOM

 var load = function (library, callback)
 * @param {String} library Relative path to a local library or URL to an external one.
 * @param {Function} callback Function to be call after library injection.

loadAll()

Injects several JS libraries in the DOM, mantainig a precedency . Precedency is assigned by the array index.

 var loadAll = function (librariesArray, callback)
 * @param {Array} librariesArray An array of libraries to inject.
 * @param {Function} callback Function to be call after library injection.
